WEEKk01 - Array practice
โดยสัปดาห์แรกเราจะทบทวนการใช้ array โดยใช้ ChatGPT เจนโจทย์เพื่อมา self-practice

// 1. เขียนฟังก์ชันที่รับ Array ของตัวเลข แล้วคืนค่าเฉลี่ย
- รวมผลบวกของตัวเลขทั้งหมดใน array
- หารด้วยจำนวนสมาชิกของ array
- คืนค่าผลลัพธ์เป็นค่าเฉลี่ย

// 2. เขียนฟังก์ชัน findMax(arr) ที่รับ array ของตัวเลข แล้ว return ค่าที่มากที่สุด
- สมมติให้ค่าตัวแรกเป็นค่ามากสุดชั่วคราว (let max = arr[0])
- วนลูปเช็คทุกตัว ถ้าเจอตัวที่มากกว่าก็อัปเดต
- คืนค่าที่มากที่สุด

// 3. เขียนฟังก์ชัน factorial(n) ที่หาค่าแฟกทอเรียล
- สูตร n! = n * (n-1) * (n-2) * ... * 1
- เขียนลูปคูณเลขไปเรื่อย ๆ

// 4. เขียนฟังก์ชันที่รับ array ของคำ แล้วคืนค่า array ใหม่ที่ตัวอักษรทั้งหมดเป็นตัวพิมพ์ใหญ่
- วนลูปผ่านทุกคำใน array
- ใช้ .toUpperCase() เปลี่ยนเป็นตัวพิมพ์ใหญ่
- เก็บไว้ใน array ใหม่แล้ว return

// 5.เขียนฟังก์ชัน unique(arr) ที่คืนค่า array ใหม่โดยลบค่าที่ซ้ำออก
- สร้าง array ใหม่ (result) ไว้เก็บค่าที่ไม่ซ้ำกัน
- วนลูปตรวจค่าทีละตัว
    - ถ้าค่ายังไม่เคยมีใน result → ให้เพิ่มเข้าไป
    - ถ้ามีอยู่แล้ว → ข้าม
- คืนค่า result ที่ไม่มีค่าซ้ำ
