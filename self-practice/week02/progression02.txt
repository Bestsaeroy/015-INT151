1. ให้รับค่ามาแล้วเช็คว่าเป็น Object or Array มั้ย ถ้าเป็นก็ให้ return true
- อย่างแรกเราจะเช็ค Array โดยใช้ .length ถ้าเท่ากับ 0 ก็เป็นค่าว่างให้ return true
- ต่อมาเราจะเช็นในส่วนของ Object โดยใช้ Object.keys(x).length ถ้าเท่ากับ 0 ก็เป็นค่าว่างให้ return true
- ส่วนค่าอื่นๆที่ไม่ใช้ให้ return false

2. โจทย์: ระบบธนาคาร (Bank Account)
   สร้าง Object account ที่เก็บ: accountNumber, owner, total
   Method:
   deposit(money) → เพิ่มเงินในบัญชี
    - โดยเราจะรับค่า money แล้วเอาไปบวกใน total
   withdraw(money) → ถอนเงิน (ถ้าเงินพอ)
    - เราจะเช็คจำนวนเงินที่เราจะถอนกับจำนวนเงินในบัญชีที่เรามีอยู่ก่อน
    - แล้วถ้าเงินพอก็เอาไปลบกับเงินในบัญชี
    - ถ้าเงินไม่พอก็ให้แสดง 'ยอดเงินไม่เพียงพอ'
   getTotal() → แสดงยอดเงินคงเหลือ
    - return this.total
